# CadastroUsuario

Este repositório contém um projeto simples de cadastro de usuário. O objetivo principal é servir como material de estudo e prática, ajudando a fixar conteúdos relacionados a desenvolvimento backend, CRUD e integração com banco de dados.

## Objetivo do projeto

O sistema foi construído com foco em aprendizado. Ele permite cadastrar usuários e manipular dados básicos, reforçando conceitos como:

- Estruturação de um projeto backend
- Organização em camadas
- Operações CRUD (Create, Read, Update, Delete)
- Persistência e gerenciamento de dados
- Boas práticas de desenvolvimento

## Tecnologias utilizadas


- .NET / C#
- Entity Framework Core
- Banco de dados relacional (PostgreSQL ou outro utilizado)
- API REST
